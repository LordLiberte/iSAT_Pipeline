# Presentación iSAT_Pipeline (15-20 min)

## 1. Introducción (2 min)
- Nombre del proyecto: iSAT_Pipeline.
- Equipo: individual.
- Problema: detección temprana de piezas defectuosas en línea de producción.
- Objetivo: pipeline DataOps con alertas en tiempo real y dashboard analítico.

## 2. Arquitectura (4 min)
- Fuente de datos sintética y ETL.
- Almacenamiento intermedio y modelo ML.
- API REST para predicción y dashboard con Streamlit.
- Orquestación con Prefect y despliegue con Docker.

## 3. Implementación (5 min)
- Demo rápida de `generator/generate.py`, `orchestrator.py` y `dashboard/app.py`.
- Snippets clave de ETL y modelo.
- Explicación de cómo se ejecuta el pipeline.

## 4. Metodología ágil (3 min)
- Product backlog y sprints.
- Uso de GitHub Projects como tablero conceptual.
- Sprint retrospective y lecciones aprendidas.

## 5. Resultados y métricas (3 min)
- Cobertura de tests y calidad del pipeline.
- Qué funciona y mejoras futuras.
- Comparación con el plan inicial.

## 6. Conclusión (2 min)
- Aprendizajes: DataOps, CI/CD, Docker, ML.
- Próximos pasos: despliegue AWS, datos reales y validación en producción.
